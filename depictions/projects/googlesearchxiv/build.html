Introducing GoogleSearchXIV, a highly customizable Google search widget for iOS.


Features Include:

Regular Options

  • Search bar color
  • Search bar text color
  • Search bar placeholder text color
  • Default or custom Google icon color
  • Default or custom mic icon color

Dynamic Options

  • Light and dark search bar colors
  • Light and dark search bar text color
  • Light and dark search bar placeholder text color
  • Default or custom light and dark Google icon color
  • Default or custom light and dark mic icon color

Global Options

  • Search bar opacity
  • Search bar blur
